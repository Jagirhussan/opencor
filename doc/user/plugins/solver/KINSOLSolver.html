The KINSOLSolver plugin uses KINSOL to solve systems of NLA equations and it can be customised through the following properties:

  • Maximum number of iterations: the maximum number of iterations used by the solver to solve the system (default: 200).
  • Linear solver: the linear solver used by the solver (default: Dense).

    Dense, Banded, GMRES, BiCGStab or TFQMR can be used.

    Banded requires specifying both an upper and a lower half-bandwidth value.

  • Upper half-bandwidth: the upper half-bandwidth value used by the Banded linear solver (default: 0).

    the upper half-bandwidth value must be between 0 and n-1 with n the number of NLA equations in the system.

  • Lower half-bandwidth: the lower half-bandwidth value used by the Banded linear solver (default: 0).

    the lower half-bandwidth value must be between 0 and n-1 with n the number of NLA equations in the system.

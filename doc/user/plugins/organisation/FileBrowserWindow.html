The FileBrowserWindow plugin offers a convenient way to access physical files, remembering the folder or file that was selected when OpenCOR was last run. By default, the home directory is selected:

To double click on a folder expands its contents, as can be seen by double clicking on the Windows directory:

On the other hand, to double click on a file results in that file being opened in OpenCOR. The rendering of the file depends on either its type (if it is recognised) or the view being currently selected.

Files can also be dragged from the File Browser window and dropped onto the File Organiser window.

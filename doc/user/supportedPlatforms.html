OpenCOR can be used on the following versions of Windows, Linux and OS X. Alternatively, you can try to build it yourself.

Windows

OpenCOR is supported on both the 32-bit and 64-bit versions of Windows XP and later.

Linux

OpenCOR 0.1.x and 0.2 are supported on both the 32-bit and 64-bit versions of Ubuntu 12.04 LTS (Precise Pangolin) and later while other versions of OpenCOR are supported on Ubuntu 14.04 LTS (Trusty Tahr) and later. In all cases, OpenCOR may also work with earlier versions of Ubuntu, as well as with other Linux distributions, though additional system libraries may be needed in the latter case.

OS X

OpenCOR 0.1.x is supported on OS X 10.8 (Mountain Lion) and later while other versions of OpenCOR are supported on Mac OS X 10.7 (Lion) and later.
